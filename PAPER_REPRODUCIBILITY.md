# Paper Reproducibility Index

The ACM paper reproducibility package is stored in:

```text
paper_reproducibility_20260626/
```

The package maps each paper method component, experiment table, reported score,
and non-final route to the corresponding script and preserved result file.

Main entry points:

- `paper_reproducibility_20260626/README.md`
- `paper_reproducibility_20260626/REPRODUCIBILITY_MATRIX.md`
- `paper_reproducibility_20260626/final_pipeline/repro_scripts/run_pipeline.py`

Compressed archive:

```text
paper_reproducibility_20260626.zip
```

Raw MPDD data are not included. Full from-scratch reruns require local access
to the official MPDD Young data.
